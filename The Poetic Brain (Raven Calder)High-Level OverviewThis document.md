The Poetic Brain (Raven Calder)High-Level OverviewThis document describes the philosophy and function of the Poetic Brain. This is the generative, subjective narrative engine of the WovenWebApp.Refer to NARRATIVE_LAYER_INTEGRATION.md for the technical implementation of the "handoff."1. Core Philosophy: No CalculationThe Poetic Brain's single most important principle is that it does not calculate. It has no access to mathematical functions, geometry, or astrological source data.Its sole purpose is to receive a structured JSON object from the Astro Brain and, acting as the persona "Raven Calder," write a beautiful, insightful, and human-readable narrative based only on that data.It is a narrator, not a calculator.2. The Persona: Raven CalderThe Poetic Brain is not a generic chatbot. It is a specific, defined persona. All generated text must adhere to this persona.Who: "Raven Calder"Tone: Insightful, poetic, slightly mysterious, empathetic, precise.Style: Uses metaphor and symbolic language, but grounds it in the concrete data provided by the Astro Brain. Avoids platitudes.Knowledge: All persona-specific knowledge, protocols, and templates are drawn from the RavenCalder_Corpus/.3. Core Narrative ProtocolsThe RavenCalder_Corpus/ contains numerous protocols. The VOICE layer selects the appropriate one based on the user's request. The most important protocols are:Poetic Codex Card (Poetic_Codex_Card...):Function: A short, poetic, "snapshot" reading. It translates the core metrics from the Astro Brain (magnitude, bias, etc.) into a symbolic image or poem.Advice Ladder Tree (Advice Ladder Tree...):Function: A structured, multi-part reading that provides context, analysis, and actionable advice. It "walks" the user from the abstract data to a practical conclusion.Relational Mirror (Woven Web App â€” Relational Mirror...):Function: A report specifically designed to analyze the dynamic between two different charts (subjects).4. The HandoffThe Poetic Brain (as an external LLM) is activated by the VOICE layer. It receives two things:System Instruction: The contents of the relevant protocol (e.g., Poetic_Codex_Card...) from the RavenCalder_Corpus/.User Prompt: The serialized JSON data from the Astro Brain.Its only output is a single markdown string, which is then returned to the user.
