# A) WOVEN MAP v6 SOLO MIRROR TEMPLATE (Depersonalized, Production-Ready)

## Format Overview

The Solo Mirror is a **modular, dual-language report** separating **frontstage (interpretation)** from **backstage (geometry & provenance)**. It is output as **markdown text**, with optional JSON backstage export for system integration.

*3‚Äì4 paragraphs blending Behavioral Anchors, Conditional Impulses, Core Pressure Patterns, and Polarity Snapshot into a cohesive narrative. Tone: conversational, grounded, testable.*
