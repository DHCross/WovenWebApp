üí° How to Access These Features

Prerequisites:

  • ‚úÖ Valid API key configured (RAPIDAPI_KEY in .env)
  • ‚úÖ Netlify functions running (not simple HTTP server)
  • ‚úÖ Complete chart data filled in
  • ‚úÖ Valid date range specified
  • ‚úÖ Successful seismograph generation

Step-by-step Process:

  1. Switch to Seismograph tab
  2. Fill in Person A data completely
  3. Set date range (use "üìÖ Use Today" for quick setup)
  4. Click "Generate Seismograph"
  5. Wait for API processing to complete
  6. Queue Analysis and Filters buttons will appear in the seismograph results

Common Issues:

  • API Error 501: Backend functions not running (need netlify dev)
  • Missing dates: Both start and end dates required
  • Invalid API key: Need valid RAPIDAPI_KEY in environment
  • No data: API returned empty results for date range
Technical Details:
- Queue Analysis button ID: queueSeismographAnalysis
- Filters button ID: toggleSeismographFilters
- Both controlled by: buildSeismographDisplay() function
- Visibility depends on: window.seismographData being populated
