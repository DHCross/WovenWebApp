### 3. ‚úÖ Color by Directional Bias (Gradient)

**Both Components:**

**Code Reference:**
```typescript
// AccelerometerScatter.tsx, lines 56-74
const getColorFromValence = (valence: number): string => {
  const normalized = (valence + 5) / 10; // [-5, +5] ‚Üí [0, 1]
  
  if (normalized < 0.5) {
    // Red to Gray (contractive to neutral)
    const t = normalized * 2;
    const r = Math.round(220 + (148 - 220) * t);
    const g = Math.round(38 + (163 - 38) * t);
    const b = Math.round(38 + (184 - 38) * t);
    return `rgb(${r}, ${g}, ${b})`;
  } else {
    // Gray to Blue (neutral to expansive)
    const t = (normalized - 0.5) * 2;
    const r = Math.round(148 - 148 * t);
    const g = Math.round(163 - 33 * t);
    const b = Math.round(184 + 62 * t);
    return `rgb(${r}, ${g}, ${b})`;
  }
};
```

**Color Mapping:**
- Directional Bias -5 (max friction) ‚Üí `rgb(220, 38, 38)` (red)
- Directional Bias 0 (neutral) ‚Üí `rgb(148, 163, 184)` (gray)
- Directional Bias +5 (max ease) ‚Üí `rgb(0, 130, 246)` (blue)

**Status:** ‚úÖ Both use gradient coloring by Directional Bias
